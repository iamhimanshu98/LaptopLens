Overview
Welcome to LaptopLens, a Price Prediction System. Our platform harnesses the power of machine learning to predict laptop prices accurately. Whether you’re a tech enthusiast, a student, or a professional, our user-friendly interface provides instant insights into laptop costs.

Technology used
HTML & CSS : The frontend is built using HTML and CSS, ensuring a visually appealing design.
Python & Flask : The backend is powered by Python and Flask.
Machine Learning : The prediction model is based on a Random Forest algorithm.

How it works
Input Data : Users provide essential details like laptop specifications, brand preferences, and market trends.
Prediction : Our machine learning model processes this data to forecast laptop prices.
Result : Instantly view the predicted prices, empowering you to make informed decisions.
